# 🚀 SIH2025 Contribution Guide

Welcome to the **SIH2025 Repository** 👋  

---

## 📂 Step 1: Clone Repository
```bash
git clone https://github.com/pirates-of-mern/sih2025.git
cd sih2025
```

---

## 🌿 Step 2: Created My Branch
  
```bash
git checkout -b suryadeep-branch
```

---

## 📝 Step 3: Created My File

```bash
echo "Hello, I am Suryadeep.This is my contribution!" > suryadeep.md
```

---

## ➕ Step 4: Add File
```bash
git add suryadeep.md
```

---

## 💬 Step 5: Commit Your Changes
```bash
git commit -m "Added personal info file: suryadeep.md"
```
---

## ☁️ Step 6: Push to GitHub
```bash
git push origin suryadeep-branch
```
---

## 🔎 Useful Commands
Check current status:  
```bash
git status
```

See your commit history:  
```bash
git log --oneline
```

Switch to another branch:  
```bash
git checkout branch-name
```
---

Happy Contributing ❤️
